# 🌦️ Weather Dashboard

A Python-based Weather Dashboard that fetches real-time weather information using the OpenWeatherMap API and visualizes the data using Matplotlib.

## 📌 Project Overview

This project demonstrates how to integrate a public API with Python and create meaningful data visualizations. It retrieves live weather data for a specified city and displays key weather metrics in both text and graphical form.

## 🚀 Features

- 🌍 Fetches real-time weather data
- 🌡️ Displays temperature
- 💧 Shows humidity
- 🌬️ Displays wind speed
- 📊 Visualizes weather data using Matplotlib
- 📡 Uses OpenWeatherMap API
- 🖥️ Simple and user-friendly interface

## 🛠️ Technologies Used

- Python
- Requests
- Matplotlib
- OpenWeatherMap API

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-dashboard.git
```

### 2. Navigate to the Project Folder

```bash
cd weather-dashboard
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

## 🔑 Configure API Key

Open `config.py` and add your OpenWeatherMap API key.

```python
API_KEY = "YOUR_API_KEY"
CITY = "Delhi"
UNITS = "metric"
```

## ▶️ Run the Project

```bash
python main.py
```

## 📊 Output

The application displays:

- City Name
- Weather Condition
- Temperature
- Humidity
- Pressure
- Wind Speed

It also generates a weather visualization dashboard using Matplotlib.
